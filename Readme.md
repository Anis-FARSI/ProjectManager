# ProjectManager
## Project architecture
```
.
├── DEBIAN
│   └── control
└── usr
    ├── bin
    │   └── project_manager.sh
    └── share
        ├── applications
        │   └── project_manager.desktop
        └── icons
            └── icon.png
```
## Install
- Download the .deb package
- Use the following command to install the package : 

```bash
sudo dpkg -i projectManager_src.deb
```
## Use
```bash
project_manager.sh
```
